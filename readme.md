# random-color-name

> Get random color name

## Install

```
$ npm install random-color-name
```

## Usage

```js
const color = require("random-color-name");

color.all;

color.random();
//=> 'Green'
```

## API

### .all

Type: `string[]`

color names in alphabetical order.

### .random()

Type: `Function`

Random color name.
